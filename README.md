# PROJECT1_Sales_Analysis-
Sales Data EDA Project - python + pandas + matplotlib 
##  Project Overview
This project analyzes retail sales data to identify sales trends,
top-performing regions, and profitable product categories using Python.

## Dataset
- Superstore Sales Dataset
- Contains order, customer, product, sales, and profit details

##  Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

##Repository Structure
- Dataset - raw dataset
- Notebook - analysis notebook
- Visuals - charts and graphs
